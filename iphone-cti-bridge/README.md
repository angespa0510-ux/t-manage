# T-MANAGE iPhone CTI Bridge (Beta)

iPhone の着信を Windows PC 経由で T-MANAGE に連携するツールです。

---

## ⚠️ 重要: これはベータ版です

Android CTI アプリと違い、**100% の検出は保証できません**。
業務クリティカルな用途には、有料プランの **Twilio 連携版** をご検討ください。

### ❌ 動作しないケース（検出漏れ）
- iPhone と Windows の Phone Link 接続が切れている
- iPhone が集中モード / おやすみモードになっている
- iPhone 連絡先に登録済みの番号（番号ではなく「名前」で通知が届くため）
- PC がスリープ / 電源オフ
- Windows の集中モードがオンになっている

### ✅ 動作するケース
- iPhone 連絡先に未登録の新規顧客からの着信
- Phone Link 接続が安定している時間帯
- 業務用 iPhone で連絡先を意図的に登録しない運用

---

## 📋 必要なもの

- Windows 10 / 11 の PC（常時起動）
- iPhone (iOS 15+)
- Python 3.10 以降
- iPhone と PC の Bluetooth ペアリング済みの **Phone Link** アプリ

---

## 🚀 セットアップ手順

### ステップ 1: Phone Link で iPhone をペアリング

1. Windows の「**スマートフォン連携**」アプリを起動
2. 「iPhone」を選択してペアリング指示に従う
3. iPhone 側で Bluetooth / 通知 / 連絡先の共有を許可
4. PC 画面右下にテスト通知が届くか確認

> 💡 **Intel Unison の方が相性が良い場合があります**
> Phone Link が不安定なら、Intel のサイトから Intel Unison をダウンロードして試してください。

### ステップ 2: Python をインストール

1. https://www.python.org/downloads/ から Python 3.10 以降をダウンロード
2. インストール時に **「Add Python to PATH」に必ずチェック**
3. インストール完了後、コマンドプロンプトで `python --version` を実行して確認

### ステップ 3: Bridge をダウンロード

T-MANAGE 管理者から配布された `iphone-cti-bridge` フォルダを任意の場所に展開します。
例: `C:\t-manage\iphone-cti-bridge\`

### ステップ 4: 設定ファイルを作成

1. フォルダ内の `.env.example` をコピーして `.env` という名前にリネーム
2. `.env` をメモ帳で開き、T-MANAGE 管理者から教えてもらった値を入力:
   ```
   SUPABASE_URL=https://xxxxxxxxxx.supabase.co
   SUPABASE_ANON_KEY=eyJhbGc...（長い文字列）
   STORE_ID=1
   ```
3. 保存して閉じる

### ステップ 5: 通知アクセス許可

1. Windows の **設定 > プライバシーとセキュリティ > 通知**
2. 「アプリや他の送信者からの通知を取得」を **オン**
3. 「通知を送信できるアプリを選択する」で Python がリストに出たら許可
   （初回実行時に自動で表示されるダイアログで「はい」を選んでも OK）

### ステップ 6: Bridge を起動

フォルダ内の `start-bridge.bat` をダブルクリック。
初回のみ依存パッケージの自動インストールが走ります（数分）。

```
============================================================
 T-MANAGE iPhone CTI Bridge (Beta)
============================================================
 Supabase URL: https://xxxxxxxxxx.supabase.co
 Store ID:     1
 Device ID:    FRONT-DESK-PC
============================================================
✅ 通知アクセス許可 OK
🎧 通知監視開始 (間隔 2.0s, store_id=1, device=FRONT-DESK-PC)
🚀 T-MANAGE iPhone CTI Bridge 起動
```

### ステップ 7: 動作テスト

1. 別の番号から iPhone に電話をかけてもらう
2. PC 右下に Phone Link の着信通知が表示される
3. Bridge 画面に `✅ 送信成功: 09012345678` と表示される
4. T-MANAGE 側でポップアップが出れば成功

### ステップ 8: PC 起動時に自動起動（推奨）

`register-startup.bat` をダブルクリックすると、次回の PC 起動時から自動的に Bridge が起動します。

## 🔧 トラブルシューティング

| 症状 | 原因と対処 |
|---|---|
| 「Python がインストールされていません」 | Python 再インストール時に「Add Python to PATH」にチェック |
| 「通知アクセスが許可されていません」 | Windows 設定 > プライバシー > 通知 で許可 |
| 着信通知は PC に届くが、Bridge が反応しない | iPhone 連絡先に登録済みの番号の可能性。未登録番号でテスト |
| 「Supabase 送信エラー」 | `.env` の URL / KEY を再確認。インターネット接続確認 |
| 着信通知自体が PC に届かない | Phone Link の再ペアリング。集中モードをオフ |
| 名前だけ表示されて番号が取れない | iPhone 連絡先から当該番号を削除する、または業務用 iPhone を連絡先未登録のまま運用する |

---

## 🔄 更新方法

新しいバージョンが配布されたら、`.env` ファイル以外を上書きコピーし、`start-bridge.bat` を再起動してください。

## 🔐 プライバシー

本ツールは Windows の通知センターに表示された通知のうち、**電話番号らしき文字列を含むもののみ** を Supabase に送信します。
SMS 本文やメール内容は送信しません。ただし Windows 通知センターに表示されるすべての通知テキストは一時的にメモリ上で処理されます。
