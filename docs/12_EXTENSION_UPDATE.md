# 12. 拡張機能 再読み込みガイド（社内PC向け）

> Chrome / Edge 拡張機能の更新反映手順。
> T-MANAGE の host_permissions が `ange-spa.t-manage.jp` に更新されたため、
> 各社内PCで拡張機能の再読み込みが必要です。

**対象**: 社内PC全台（事務所、各店舗）
**所要時間**: 1台あたり 約5分
**実施期限**: 本番運用開始前（2026/6/1 まで）

---

## 🎯 今回の更新内容

```
旧:
  host_permissions: ["https://t-manage.vercel.app/*"]

新:
  host_permissions: ["https://ange-spa.t-manage.jp/*"]
```

**影響**: 拡張機能を再読み込みしないと、T-MANAGE との連携が止まる可能性があります。

---

## 📦 対象の拡張機能（3つ）

| 拡張機能名 | バージョン | 用途 |
|---|---|---|
| T-MANAGE 連携拡張（Chrome） | v2.2.0 | 通話記録・CTI連携 |
| T-MANAGE SMS拡張（Edge） | v1.1.0 | SMS自動送信 |
| T-MANAGE エステ魂拡張（Chrome） | v2.4.0 | エステ魂スクレイピング |

---

## 🔧 再読み込み手順

### Chrome 拡張機能の場合

#### 手順1: 拡張機能ページを開く

**方法A**: アドレスバーに入力
```
chrome://extensions/
```

**方法B**: メニューから
```
Chrome右上の ⋮ メニュー
  → 拡張機能
  → 拡張機能を管理
```

#### 手順2: デベロッパーモードをON

```
右上の「デベロッパーモード」トグルを ON
```

#### 手順3: 該当拡張機能の更新

```
「T-MANAGE 連携拡張」を探す
  ↓
  「🔄 更新」または「読み込み直す」ボタンをクリック
  
  または
  
  一旦「削除」→ 再度「パッケージ化されていない拡張機能を読み込む」
  → t-manage リポジトリ内の chrome-extension フォルダを選択
```

### Edge 拡張機能の場合

#### 手順1: 拡張機能ページを開く

```
edge://extensions/
```

#### 手順2: 該当拡張機能の更新

```
「T-MANAGE SMS拡張」を探す
  ↓
  🔄 アイコンで再読み込み
```

## 🔍 反映確認方法

### 1. 拡張機能のバージョン確認

```
chrome://extensions/
  → 該当拡張機能のカードを表示
  → バージョン番号を確認

期待値:
  T-MANAGE 連携拡張       v2.2.0
  T-MANAGE SMS拡張        v1.1.0
  T-MANAGE エステ魂拡張   v2.4.0
```

### 2. 動作確認

```
1. https://ange-spa.t-manage.jp/ を開く
2. 拡張機能のアイコンをクリック
3. ポップアップが正常に表示される
4. 「連携中」「接続済み」等の表示を確認
```

## ❓ トラブルシューティング

### Q1: 拡張機能アイコンが消えた

```
対処:
  1. chrome://extensions/ を確認
  2. 無効化されていないか確認
  3. ピン留めされているか確認（ツールバー右端のパズルアイコン）
```

### Q2: 「T-MANAGE に接続できません」エラー

```
原因: 古い host_permissions のまま
対処:
  1. 拡張機能を完全削除
  2. 最新版を再インストール
  3. ブラウザ再起動
```

### Q3: SMS送信できない

```
原因: Edge拡張の host_permissions 未更新
対処:
  1. edge://extensions/ を開く
  2. T-MANAGE SMS拡張 を再読み込み
  3. 一度ログアウト → 再ログイン
```

### Q4: エステ魂の自動取得が動かない

```
原因: エステ魂拡張のpermission未更新
対処:
  1. chrome://extensions/ を開く
  2. T-MANAGE エステ魂拡張 を削除
  3. estama-extension フォルダから再インストール
```
