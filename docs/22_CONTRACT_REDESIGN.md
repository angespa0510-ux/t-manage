# ③業務委託契約書 — 再設計書

**作成日**: 2026-04-28
**目的**: 「施術業」として税務上一貫した契約書文言に整備
**前提**: `21_TREATMENT_BUSINESS_POSITIONING.md` に従う
**現行**: `app/contract-sign/[token]/page.tsx`（v1〜既存契約者署名済）
**新版予定**: v2（弁護士レビュー前提）

---

## 1. 設計方針

### 1.1 現行契約書の評価
**良い点（残す）**:
- 18条構成で必要な論点はカバー
- 第5条 風俗行為の禁止
- 第8条 引き抜き禁止（2年間）
- 第6条 肖像権の取り扱い

**改善点**:
- 第1条 業務内容に「施術技術の提供」が明記されていない
- 第5条 禁止事項に **酒類提供・同伴・アフターの明示禁止が不足**
- カルテ記録義務・研修受講義務が不在
- 「業務委託」性を補強する裁量条項が弱い
- 税務責任（受託者の確定申告責任）が明記されていない

### 1.2 設計の柱
1. **施術技術提供契約** として表題・前文・第1条を再構成
2. **禁止事項を厚く**：飲酒・同伴・アフター・店外接触の全面禁止
3. **施術業の実態証跡**：カルテ記録義務・研修受講義務を明記
4. **業務委託性の補強**：受託者の裁量・自己責任を明確化
5. **税務責任の明記**：源泉徴収なしの根拠と確定申告責任

## 2. 新契約書 構成案（v2）

### 表題
```
業務委託契約書（施術技術提供契約）
```

### 前文
```
合同会社テラスライフ（屋号：Ange Spa。以下「甲」という）と、
受託者（以下「乙」という）は、甲が運営する店舗における
リラクゼーション施術業務（ボディケア・アロマトリートメント・
リンパケア等）の業務委託に関して、以下の通り契約を締結する。

本契約は、乙が独立した個人事業主として、
甲に対し施術技術を提供する業務委託契約であり、
雇用関係を生じさせるものではない。
```

---

### 第1条（業務の内容）— 改訂
```
乙は、甲の委託を受け、以下の業務を行うものとする。

1. リラクゼーション施術業務
   （ボディケア・アロマトリートメント・リンパケア・
    指圧・スウェディッシュ等の施術技術の提供）
2. 施術前の体調・希望のカウンセリング
3. 施術内容のカルテ記録
4. 顧客からの代金の受領、代金の管理
5. 乙が使用した施術ルームおよび備品の清掃、整理整頓
6. 店舗運営に付随する事務連絡への対応
7. 前各号に付随する一切の業務
```

---

### 第2条（業務の実施・受託者の裁量）— 改訂
```
1. 乙は、自己の独立した事業として、自己の裁量により業務を
   遂行するものとする。
2. 乙は、自己の判断により、業務日時・出勤頻度を設定し、
   甲は乙に対し業務時間や勤務日数を強制しない。ただし、
   一度確定した予約またはシフトに関しては、正当な理由なく
   中止・遅延してはならない。
3. 乙は、提供する施術コースの種類について、自己の技術
   習熟度に応じて選択する裁量を有する。
4. やむを得ぬ理由で業務に従事できない場合は、直ちに甲に
   通知するものとする。
```

→ 業務委託性を強く打ち出す（雇用と区別）

---

### 第3条（委託料の支払いおよび税務）— 改訂
```
1. 甲は、乙に対し、当日の業務終了後に規定の委託料を
   現金または振込にて支払うものとする。委託料は、
   乙が提供した施術コースごとに算定する技術提供報酬とする。
2. 本契約に基づく報酬は、所得税法上、源泉徴収の対象と
   なる業務に該当しないため、甲は乙の報酬から所得税の
   源泉徴収を行わない。乙は、自らの責任において確定申告を
   行うものとする。
3. オプション料金等に関しては、乙の自己管理とし、甲は
   これに関与せず、一切の責任を負わない。
4. 報酬額は消費税相当額を含むものとする。ただし、乙が
   適格請求書発行事業者でない場合、甲が被る消費税の
   仕入税額控除の制限相当額について、甲乙協議の上、
   報酬額を調整できるものとする。
```

→ **第2項が最重要**：源泉徴収なしの根拠と乙の確定申告責任を明示

---

### 第4条（費用負担）— 微改訂
```
業務遂行に要する消耗品（オイル、タオル、リネン等）の
費用、および店舗設備の使用料については、別途甲乙間で
定める規則に従うものとする。乙の業務遂行のために
甲が提供する備品・リネン代は、委託料から控除する形で
精算する。
```

→ 既存の welfare_fee = 備品・リネン代の運用を明記

---

### 第5条（禁止事項）— 大幅改訂
```
乙は、以下の行為を行ってはならない。

1. 施術業務において、風俗行為・売春行為を行うこと、
   またはそれを助長すること。
2. 宣伝・集客活動において、風俗行為を想起させる文言・
   画像を使用すること。
3. 店舗内において、酒類の提供を受けること、または
   顧客と共に飲酒すること。
4. 顧客との店外での飲食、同伴、送迎、アフター
   （業務時間外の私的な交際）を行うこと。
5. 業務時間外に、顧客と私的な連絡（電話・SNS・
   メッセージ等）を取ること。
6. 甲の許可なく、同業他社との兼業（掛け持ち）を
   行うこと。
7. 本契約に基づき知り得た顧客との私的な接触、
   私的な金銭の授受、または他社への勧誘。
8. 甲の名誉や信用を毀損する行為。
9. 施術カルテへの記録を怠ること、または虚偽の記録を
   行うこと（第○条参照）。
```

→ **3, 4, 5項が新規追加**：施術業として最も重要な「ホステス相当」回避条項

### 【新設】第10条（施術技術の研修受講義務）
```
1. 乙は、甲が定める施術技術研修プログラムを受講し、
```

### 【新設】第11条（施術カルテの記録義務）
```
1. 乙は、施術業務の遂行にあたり、施術前のカウンセリング、

2. 当該カルテは、施術品質の維持・顧客満足度の向上・

3. カルテの記録は、施術業務の一部として委託料に
   含まれる。
4. 乙は、虚偽の記録、または記録の意図的な省略を

```

→ 施術業の実態証跡として最重要

## 3. システム実装上の対応

### 3.1 契約書バージョン管理
- `CURRENT_CONTRACT_VERSION` を v1 → v2 に更新
- 既存セラピスト（v1署名済）と新規セラピスト（v2署名）を分離
- DB: `contracts.version` で識別

### 3.2 既存契約者の再締結プロセス
**選択肢A：強制的な再締結**
- 全員に対して v2 への再署名を要求
- 6/1までに完了

**選択肢B：段階的な再締結**（推奨）
- 新規入店者は v2 から
- 既存者は順次 v2 に再署名（6月中目標）
- v1の有効性は維持（過去分の保護）

→ 選択肢Bを推奨：セラピストの動揺を最小化し、説明会の余裕を持たせる

### 3.3 v2 同意フロー
```
セラピストマイページ
  → ログイン後、「契約書 v2 への同意が必要」表示
  → 契約書本文を表示
  → 主な変更点をハイライト表示
    - 施術技術研修の受講義務（新設）
    - 施術カルテ記録義務（新設）
    - 酒類・同伴・アフターの全面禁止（明文化）
  → 「内容を確認の上同意する」チェックボックス
  → 電子署名（手書き）
  → 同意後は通常のマイページに遷移
```

### 3.4 弁護士レビュー前のフロー

## 4. 弁護士レビュー時の論点メモ

レビュー依頼時に弁護士に確認すべき項目：

1. **業務委託契約 vs 雇用契約の境界**
   - 第2条の裁量規定で「業務委託」性を担保できているか
2. **税務責任条項の有効性**
   - 第3条第2項の文言で源泉徴収なしの根拠として通用するか
3. **禁止事項の妥当性**
   - 第5条の店外接触禁止が個人の自由を過度に制限していないか
4. **研修・カルテ義務の妥当性**
   - 第10条・第11条が業務委託の範囲内に収まるか
5. **既存契約者の扱い**
   - v1署名済との並行運用で法的リスクがないか
