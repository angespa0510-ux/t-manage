# 🏷 Part 2: ロゴ制作プロンプト

## 🎯 ロゴの方針

**Ange Spa** の "Ange" はフランス語で「天使」、"Spa" は「癒し」。
ブランドは「清楚・上品・静けさ・柔らかい女性らしさ」。

### デザインの基本方針

- ✅ ミニマル・上品（仕様 ■20 に準拠）
- ✅ 明朝系セリフフォント中心（現サイトの Cormorant Garamond と調和）
- ✅ 単色で完結（印刷・箔押し・刺繍等の二次利用対応）
- ✅ 透過背景の PNG（もしくは SVG）で書き出し
- ❌ 極端に女性的 / フリル / キャラクター風は避ける
- ❌ 複雑な装飾 / 3D / グラデーションは避ける

### カラー展開

1. **メイン**: ピンク `#E8849A`
2. **モノクロ**: ダークグレー `#2B2B2B`
3. **反転**: ホワイト `#FFFFFF`（ダーク背景用）
4. **箔押し風**: ローズゴールド（印刷物用・参考）

---

## 🎨 ロゴ案（5方向）

現在HPで使用している「Cormorant Garamond で "Ange Spa"」を**ベース**に、5つの方向性のプロンプトを用意。

### 案A（推奨）: ミニマル ワードマーク＋細い装飾

**方向性**: 文字だけだが、上下の細い罫線で上品に整える。現状のHPトーンに最も合う。

```
A minimal elegant logo design featuring the text "Ange Spa" in a refined
serif typeface similar to Cormorant Garamond or Didot. The text is
centered, with thin horizontal lines (1px) above and below, creating a
refined frame. Below the main text, a small French subtitle "Healing &
Relaxation" in a thin sans-serif or serif italic. Dusty pink color
(#E8849A) on transparent background. Ultra-minimal, sophisticated,
luxurious Japanese-French spa aesthetic. High-end boutique brand logo.
Vector-style, clean edges, no gradient, no 3D. Square 2048x2048, logo
centered with generous white space around.

no watermarks, no extra text, no decorative flourishes, no cartoon,
no 3D effect, no shadow, single color only
```

### 案B: 翼のモチーフ（Ange = 天使）

**方向性**: ブランド名の「Ange（天使）」から発想。抽象化した片翼 or 1対の細い翼。

```
A minimal elegant logo design: an abstract single angel wing rendered
in a single thin continuous line (1-2px stroke), geometric and refined,
not illustrative. Below the wing, the text "Ange Spa" in a thin serif
typeface like Cormorant Garamond. Balanced vertical composition. Dusty
pink color (#E8849A) on transparent background. Luxurious high-end
boutique brand aesthetic, Japanese-French refinement. Minimal, line-art
style, no fill, just elegant strokes. Square 2048x2048.

no humans, no complex feather details, no 3D effect, no gradient,
no extra text, no cartoon
```

### 案C: 花のモチーフ（牡丹 or 椿）

**方向性**: 静物画像でも多用している牡丹・椿などのフラワーアイコン。

```
A minimal elegant logo design: an abstract single peony flower rendered
in a single continuous thin line (1-2px stroke), geometric and simplified,
viewed from above. Very delicate, like a Japanese seal design. Below the
flower, the text "Ange Spa" in a thin serif typeface. Centered vertical
composition. Dusty pink color (#E8849A) on transparent background.
Luxurious high-end boutique brand logo, Japanese botanical refinement.
Line-art style, single stroke, no fill. Square 2048x2048.

no complex petal details, no 3D, no gradient, no extra text, no cartoon
```

### 案D: モノグラム「A S」

**方向性**: A と S を組み合わせた、ホテル・高級ブティック風モノグラム。

```
A minimal elegant monogram logo design combining letters "A" and "S"
in a refined serif typeface like Didot or Cormorant Garamond. The letters
overlap slightly or share a common vertical stroke, forming a unified
monogram mark. Enclosed in a thin circle or thin-line rectangle frame.
Below the monogram, small text "Ange Spa" in thin serif. Dusty pink
color (#E8849A) on transparent background. Classical high-end hotel /
boutique brand aesthetic, French-Japanese refinement. Flat vector style.
Square 2048x2048.

no 3D, no gradient, no shadow, no extra decorative elements, no cartoon
```

### 案E: 光のモチーフ（天使の輪）

**方向性**: Ange = 天使から、光の輪 / 細い円の装飾。

```
A minimal elegant logo design: a single thin circle (1-2px stroke) as
the primary symbol, with a subtle break at the top representing a soft
halo or ring of light. Centered inside the circle, the text "Ange Spa"
in a refined thin serif typeface like Cormorant Garamond. Dusty pink
color (#E8849A) on transparent background. Ultra-minimal, ethereal,
celestial, Japanese-modern boutique aesthetic. Single color, line art,
no fill. Square 2048x2048.

no 3D, no gradient, no complex pattern, no extra text, no cartoon
```

## 🛠 ロゴ生成の順序

### ステップ1: 複数案を生成して比較
上記 A〜E のプロンプトを **Gemini / Imagen で1〜2枚ずつ生成** して、方向性を決定。

### ステップ2: 決めた案のバリエーション展開
選んだ案で以下を生成して揃える：

- **ピンク版**（メインカラー）
- **モノクロ黒版**（書類・名刺用）
- **モノクロ白版**（ダーク背景用）
- **横型バージョン**（ヘッダー用、幅広）
- **縦型バージョン**（スクエア SNS 用）
- **アイコン単体**（ファビコン・SNSアイコン用、文字なし）

### ステップ3: HPに組み込み
- **SVG化**（推奨）: Adobe Illustrator / Inkscape / Figma でベクター化
- **PNG透過**: そのまま使う場合

配置先:
- `public/logo/ange-spa-main.svg`（メインロゴ）
- `public/logo/ange-spa-icon.svg`（アイコン）
- `public/logo/ange-spa-mono.svg`（モノクロ）
- `public/favicon.ico`（ブラウザタブ用）

HPのヘッダー・フッターのテキストロゴを SVG ロゴに差し替えます（コード対応）。

## 💡 ロゴ制作の別アプローチ

Gemini で生成しつつ、**プロのデザイナーに依頼** するのも良い選択肢:

| サービス | 料金目安 | 特徴 |
|---|---|---|
| **ココナラ** | 3,000〜30,000円 | 豊富な実績、選べるデザイナー |
| **クラウドワークス** | コンペ形式 5,000円〜 | 複数案から選べる |
| **Looka** (AI) | 月額 $20 程度 | AIで数分で生成 |
| **Brandmark.io** | $25〜 | 商用利用OK、即完成 |
| **Canva Pro** | 月額 $13 | テンプレートから調整 |

本格展開するなら**プロ依頼 + SVGデータ納品**がおすすめ。
Gemini は「方向性を確定する叩き台」として使うのが効率的です。
