## 👤 人物について

仕様上、人物写真はセラピスト本人撮影のみが原則。AI生成で人物を入れる場合は：
- 顔がはっきり見えない（横顔・後ろ姿・シルエット）
- 「モデル」ではなく「ブランドイメージ」として成立
- 個人を特定できない構図

これらに該当する構図のみ採用可能。本書では人物あり/なしを明記。

# 📝 生成のコツ

## プロンプトの先頭に入れると品質UP

```
Consistent with a dreamy pastel pink sunset rooftop aesthetic featuring
magical bokeh light particles, warm golden hour atmosphere, soft editorial
Japanese photography style.
```

これを既存プロンプトの冒頭に足すと、ブランドトーンの一貫性が大幅にアップします。

## 季節替えバリエーション

同じ構図で季節を変える場合：

- **春**: `spring afternoon, cherry blossom petals falling`
- **夏**: `summer evening, warm breeze, sheer fabric`
- **秋**: `autumn golden hour, maple leaves, amber tones`
- **冬**: `winter dusk, soft snow, frosted window`

## ポジティブに効く語

```
editorial, cinematic, dreamy, ethereal, magical,
soft diffused light, shallow depth of field,
golden hour, sunset, pastel,
refined, elegant, understated luxury,
Japanese-modern aesthetic
```

## アウトには効く語（避ける）

```
girly, cute, sweet, frilly, lace-heavy,
saturated, flashy, neon,
anime, cartoon, 3D render,
glamorous, seductive, provocative
```

# 🔄 画像配置後の使い方

1. Gemini / Google AI Studio でプロンプト実行
2. 生成画像をダウンロード
3. ファイル名を表のとおりに揃える
4. `public/images/placeholder/` に配置
5. git commit & push で Vercel 即反映

動画導入時は HOME の `<Image>` を `<video autoplay loop muted playsinline>` に差し替える調整が別途必要（コード側で対応します）。
