**作成日**: 2026-04-24
**移行完了目標**: 6/1本番運用開始後、順次判断

## 1. 背景と目的

### 1.1 現状

```
旧HP:  ange-spa.com
  ├─ 運用業者: Panda Web Concierge
  ├─ ドメイン: 業者名義の可能性大
  ├─ サーバー: Kagoya（ns0.kagoya.net, ns1.kagoya.net）
  ├─ 管理画面: ange-spa.com/pwc-admin
  └─ 月額: 業者に保守費支払い中
  
問題点:
  ❌ 業者に移管不可と言われた
  ❌ HP内容の変更が業者依存
  ❌ T-MANAGEとの連携が困難
  ❌ 月額固定費が発生
```

### 1.2 解決策：完全自社化

```
新HP:  ange-spa.jp（2026-04-24 取得）
  ├─ ドメイン: 合同会社テラスライフ名義
  ├─ サーバー: Vercel（自社管理）
  ├─ 管理: T-MANAGE 統合
  └─ 月額: ほぼゼロ（ドメイン代年約¥3,124のみ）
```

## 6. メール移行計画

### 6.1 現状

```
info@ange-spa.com
  ↓
  業者管理 or 共用サーバー（Kagoya？）
  ↓
  受信先の確認必要
```

### 6.2 移行案

#### 案A: Gmail で独自ドメインメール（推奨）

```
Google Workspace 契約
  ↓
  info@ange-spa.jp
  ↓
  月額 ¥1,360/ユーザー
  （Business Starter プラン）

メリット:
  ✅ プロ仕様、安定性最高
  ✅ Googleドライブ・カレンダー・Meet付き
  ✅ T-MANAGE（Gmail API）との相性◎
```

#### 案B: Cloudflare Email Routing（無料）

```
ange-spa.jp → info@ange-spa.jp
  ↓
  個人のGmailに転送
  ↓
  送信は個人Gmail or SMTP設定

メリット:
  ✅ 無料
  ✅ 設定簡単（5分）
  
デメリット:
  ⚠️ 送信時の設定が煩雑
  ⚠️ info@ange-spa.jp から直接送信は別途設定
```

#### 案C: お名前.comメール（安い）

```
お名前.com のメールサービス契約
月額 ¥110〜

メリット:
  ✅ 安価
  ✅ お名前.com で一元管理

デメリット:
  ⚠️ 機能・使い勝手は限定的
```

### 6.3 推奨

**当面は案B（Cloudflare Email Routing 無料）** で info@ange-spa.jp を運用し、
成長に応じて **案A（Google Workspace）** へアップグレードが現実的。
